- ## open in browser
	- > 用浏览器打开文件
	- `alt + b` 用默认浏览器打开
- ## 格式化工具
	- ### Prettier
		- > 代码格式化
		- 添加如下配置：
			- ```json
			  {
			    // 设置默认格式化工具
			    "editor.defaultFormatter": "esbenp.prettier-vscode",
			    // 设置保存时自动格式化
			    "editor.formatOnSave": true
			  }
			  ```
