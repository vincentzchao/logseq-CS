- ## 什么是 Redis Query Engine
	- Redis Open Source , Redis Enterprise Software 和 Redis Cloud 都具备的一个功能.
	- 用于搜索和查询 Redis 数据.
	- 需要安装 [[RediSearch]] 模块以使用此功能.
