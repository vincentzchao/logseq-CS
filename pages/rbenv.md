- ## 什么是 rbenv
	- 即 the Ruby version manager
	- 用于管理和切换不同的 Ruby 版本.
- ## 安装 rbenv
	- 执行 `curl -fsSL https://rbenv.org/install.sh | bash` 进行安装 (脚本内部还是使用的 Homebrew 进行安装的)
- ## 安装 Ruby
	- 执行 `rbenv install -l` 查看有哪些稳定版本的 Ruby
