- ## 创建项目
	- ``` sh
	  # 创建项目
	  npm create astro@latest
	  
	  # 启动服务
	  npm run dev
	  
	  # 生成静态网页，在 /dist 目录下
	  npm run build
	  
	  # 预览 /dist 目录下的静态网页
	  npm run preview
	  ```
- ## 开发环境
	- 可以使用 Astro 提供的编辑器插件: [Editor Setup](https://docs.astro.build/en/editor-setup/)
