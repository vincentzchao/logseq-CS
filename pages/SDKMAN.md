- ## 一句话解释
	- 一个可以管理 JDK 、Tomcat 等多种软件版本的
- ## 安装
	- [SDKMAN Install](https://sdkman.io/install)
- ## 命令速查
	- ``` sh
	  # 查看命令帮助
	  sdk help
	  
	  # 查看子命令帮助
	  sdk help <subcommand>
	  
	  # 查看某种 sdk 可用版本 如 java
	  sdk list <sdk>
	  ```
