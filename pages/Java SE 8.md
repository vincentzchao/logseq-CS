	- ### JDK 与 JRE
		- `JDK` (Java SE Development Kit)
		- `JRE` (Java SE Runtime Environment)
		- `JDK` 是 `JRE` 的超集 (superset)。
		- JDK = JRE + Tools (包含 compilers, debuggers 等工具)
