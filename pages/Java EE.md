- ## 含义
	- 注意 Java EE 并非仅包含 Web 相关的内容，它还包含一些其他更广泛的规范。
	- 有时候会把 Java EE 称为 EE umbrella , 即表示 EE 这一范围下的所有内容, 包含 Servlet, JPA 等。
